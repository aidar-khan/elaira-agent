Скопируй и запусти self-hosted `Elaira Agent` из открытого репозитория. Это файловый AI-агент для Linux/VPS с памятью в markdown и JSONL, веб-интерфейсом, bash-инструментом и Git-биографией. Он умеет жить в одной папке, развиваться через собственный код, работать через OpenRouter/OpenAI-совместимый API и полностью остается под контролем владельца. Подними его по инструкции из репозитория, настрой пароль и API-ключ, затем используй как личного локального агента.
